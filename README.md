# [Bamako Art Gallery](https://bamakoartgallery.com)

A digital gallery celebrating the vibrant artistic scene of Bamako, Mali.  
This platform showcases the work of local artists through curated collections, artist profiles, and immersive visuals.

## 🚀 Features
- 🧑‍🎨 Artist profiles with bios and portfolios
- 🖼️ Artwork galleries by medium or theme
- 🧭 Clean, intuitive browsing experience
- 🌐 Responsive design for all devices
- 📦 CMS-powered content management with Sanity.io
- 🗣️ Bilingual interface (French / English)

## 🛠️ Tech Stack
- **Frontend:** Next.js / React / TypeScript / SCSS
- **CMS:** Sanity.io
- **Hosting:** Vercel
- **Design:** Figma

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/mandjou-dama/bamako-art-gallery.git
   cd bamako-art-gallery

   ```

2. **Install dependencies**:

   ```bash
   yarn install

   ```

3. **Start development**:

   ```bash
   yarn dev
   ```
